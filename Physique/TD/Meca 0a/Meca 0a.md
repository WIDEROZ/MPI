#### PFD
On fait un PFD : 
$$m\vec{a} = m\vec{g}$$
On projette sur $z$ : 
$$a_{z} = -g$$
On intègre : 
$$v_{z}(t) = v(t) = -gt + v_{0} = -gt$$
car le mvt est sur la droite de vecteur directeur $\vec{e}_{z}$

On intègre encore : 
$$z(t) = -gt^{2} + H$$
On touche le trampoline quand : 
$$z(t) = 0$$
Alors, 
$$t_{1} =  \sqrt{ \frac{2H}{g} }$$
Ainsi, 
$$v(t_{1}) = -\sqrt{ 2gH }$$
